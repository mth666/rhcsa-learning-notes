# Scripting
The shebang line `#!/bin/bash` - first line, always. this tells the system which interpreter to use. Scripts without having it, the system doesn't know what to do with the file.
The commands - all the usual commands like `echo, find, grep, cp` all of it works the same inside a script.
Executable permission - the file needs `+x` or the system refuses to run it as a program.
And everything else - variables, loops, conditionals are just building on top of the above three things.
